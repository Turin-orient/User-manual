# Who Should Use Neurond Assistant?

Neurond Assistant is designed for everyone in your organization. This page helps you understand how different roles can benefit from using NA.

## For All Employees

**If you ever need to:**
- Find company information quickly
- Process documents or forms
- Write professional communications
- Get answers to workplace questions

**Then Neurond Assistant is for you!**

:::tip No Technical Skills Required
You don't need to be tech-savvy to use Neurond Assistant. If you can use email or search Google, you can use NA.
:::

## Administrator vs Regular User

### Regular Users (Most Employees)

**What you can do:**
- ‚úÖ Ask questions and get answers
- ‚úÖ Upload and process documents
- ‚úÖ Create content and drafts
- ‚úÖ View your conversation history
- ‚úÖ Export results

**What you cannot do:**
- ‚ùå Manage other users
- ‚ùå Change system settings
- ‚ùå Add or modify API keys
- ‚ùå Access system administration

**Your focus:** Using NA to complete your daily tasks more efficiently.

---

### System Administrators

**Everything regular users can do, PLUS:**
- ‚úÖ Add, edit, and remove user accounts
- ‚úÖ Manage permissions and roles
- ‚úÖ Configure API keys (OpenAI, Azure, Google)
- ‚úÖ Monitor system usage and performance
- ‚úÖ Configure workspace settings
- ‚úÖ Access system health and status information

**Your focus:** Keeping the system running smoothly and managing user access.

:::info For Administrators
If you're a system administrator, see the complete [Admin Guide](../admin-guide/admin-dashboard.md) for detailed instructions on system management.
:::

## Not Sure If NA Can Help You?

Ask yourself:
1. ‚úì Do I spend time searching for information in company documents?
2. ‚úì Do I manually type data from one document to another?
3. ‚úì Do I write similar emails or documents repeatedly?
4. ‚úì Do I need to summarize long reports or meeting notes?
5. ‚úì Do I wish I had a faster way to find company procedures?
