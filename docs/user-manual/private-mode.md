# Private Mode

## 1. Overview

**Private Mode** is a specialized workspace that allows users to interact with AI securely and confidentially. Key benefits include:

- **No History**: Conversations are not saved to your history list.
- **Enhanced Privacy**: Ideal for handling sensitive or proprietary content.
- **Safe Environment**: Suitable for confidential tasks, testing, or internal discussions.

---

## 2. Enabling Private Mode

### 2.1. Activation

- You can enable **Private Mode** directly from the system interface.
- Once activated, the system switches to the private workspace environment.

![Private Mode Enabled](/img/ui/private-mode/enabled.png)

---

## 3. Private Mode Behavior

When Private Mode is active:
- **Chat content is NOT saved to History**: Your messages and the Assistant's responses are temporary.
- **Isolated Workspace**: Conversations do not appear in your historical chat list on the sidebar.
- **Privacy Assurance**: Data shared in this mode is not used for training purposes or behavior analysis.

![Private Mode Indicator](/img/ui/private-mode/indicator.png)

---

## 4. Exiting Private Mode

### 4.1. Exit Action

- To return to standard mode, click the **X** icon or toggle the switch.
- The system will exit the private environment and return to your standard workspace.

![Exit Private Mode Button](/img/ui/private-mode/exit.png)

---

## 5. History Policy

### 5.1. Chat History Handling

- **In Private Mode**:
  - ❌ **Not saved to Chat History**.
  - ❌ **Irrecoverable**: Once you exit the mode or refresh, the conversation is lost.
- **After Exiting**:
  - Standard behavior resumes. New conversations will be saved and managed as usual.

![Private Mode Policy](/img/ui/private-mode/indicator.png)

---

## 6. Use Cases

Private Mode is particularly suitable for:
- **Sensitive Data**: Processing documents or data that should not be stored.
- **Internal Drafts**: Drafting confidential internal communications.
- **Prototyping**: Testing new prompts or temporary ideas without cluttering your history.
- **Confidential Inquiries**: Sharing information that doesn't require long-term tracking.

---

## 7. Notes & Best Practices

- **Check the Indicator**: Always verify that the **Private Mode indicator** is visible before entering sensitive information.
- **No Residual Tracking**: Avoid using Private Mode for conversations that you may need to audit or reference later.
- **Consistent Permissions**: Private Mode does not change your existing user roles or data access levels.

---

## Summary

Private Mode ensures:
- **Full Privacy**: Complete confidentiality for your AI interactions.
- **Maximum Security**: A protected environment for enterprise-grade use cases.
- **Safe Exploration**: Experiment freely without leaving a digital footprint.
