# Prerequisites

What you need before deploying Neurond Assistant.

## Purpose

Ensure you have everything ready for a smooth deployment.

## Who Should Complete This

✅ System Administrators preparing for deployment

---

## Required Accounts

### 1. Git Repository Account

**Options:**
- **GitHub** (recommended) - github.com
- **GitLab** - gitlab.com
- **Bitbucket** - bitbucket.org

**What you'll do:**
- Store Neurond Assistant code
- Connect to Vercel for automatic deployment

**Sign up:** Free account sufficient

---

### 2. Vercel Account

**Website:** vercel.com

**What it does:**
- Hosts your Neurond Assistant application
- Provides deployment infrastructure
- Handles SSL certificates automatically

**Sign up:**
1. Go to vercel.com
2. Sign up with GitHub account (easiest)
3. Or use email

**Cost:** Free tier works for small teams

---

### 3. AI Provider Account

**Choose ONE:**

**Option A: OpenAI**
- Website: platform.openai.com
- Models: GPT-4, GPT-3.5
- Pricing: Pay-per-use
- Setup: Create account → API keys

**Option B: Azure OpenAI Service**
- Website: azure.microsoft.com
- Models: GPT-4, GPT-3.5 (via Azure)
- Pricing: Enterprise agreements available
- Setup: Requires Azure subscription

**Option C: Google AI (Gemini)**
- Website: ai.google.dev
- Models: Gemini Pro
- Pricing: Competitive pricing
- Setup: Google account → API access

---

## Technical Information to Gather

### API Credentials

**From your AI provider, collect:**

**For OpenAI:**
- ✓ API Key (starts with `sk-`)
- ✓ Organization ID (if applicable)

**For Azure:**
- ✓ API Key
- ✓ Endpoint URL (e.g., `https://yourname.openai.azure.com/`)
- ✓ Deployment Name
- ✓ API Version

**For Google:**
- ✓ API Key
- ✓ Project ID

---

### Custom Domain (Optional)

**If you want custom URL:**
- Domain name registered (e.g., `assistant.yourcompany.com`)
- Access to DNS settings
- Domain provider login credentials

**Examples:**
- GoDaddy
- Namecheap
- Cloudflare
- Your company's domain registrar

---

## Software Requirements

### Your Computer

**Operating System:**
- Windows 10+
- macOS 10.14+
- Linux (any recent version)

**Web Browser:**
- Chrome (recommended)
- Firefox
- Safari
- Edge

**No special software needed!** Everything is managed through web dashboards.

## Permissions and Access

**Make sure you have:**
- ✅ Authority to create cloud accounts
- ✅ Company credit card for AI provider (costs apply)

- ✅ Access to configure DNS (if using custom domain)

## Pre-Deployment Checklist

Before proceeding to deployment:

- [ ] Git repository account created
- [ ] Vercel account created and verified
- [ ] AI provider account created (OpenAI/Azure/Google)
- [ ] API key generated and saved securely
- [ ] Neurond Assistant source code obtained
- [ ] Code pushed to Git repository
- [ ] (Optional) Custom domain ready
- [ ] Stakeholders informed about deployment

## Estimated Costs

**One-time:**
- $0 - Setup is free

**Monthly:**
- Vercel: $0-$20 (free tier sufficient for small teams)
- AI Provider: $50-$2000+ (depends on usage)
- Domain: $10-$20/year (optional)

**Total estimated:** $50-$500/month for typical small-medium deployment
