## Common Export Scenarios

### Export Data Table to Excel

**Situation:** NA created a table from invoices

**Steps:**
1. Click export icon on the table message
2. Choose **.xlsx** format
3. Download file
4. Open in Excel
5. Data is ready to use!

[Screenshot: Excel file opened with exported data]

---

### Save Email Draft

**Situation:** NA drafted an email

**Steps:**
1. Click copy icon on the email draft
2. Open your email client (Outlook, Gmail)
3. Paste into new email
4. Review and send

---

### Create Report from Summary

**Situation:** NA summarized a long document

**Steps:**
1. Export as **Word (.docx)**
2. Open in Microsoft Word
3. Add company header/footer
4. Format as needed
5. Save or print

## Export Formats Explained

| Format | Best For | Opens In |
|--------|----------|----------|
| **PDF** | Sharing, archiving, printing | Any PDF reader |
| **Word (.docx)** | Further editing, formatting | Microsoft Word, Google Docs |
| **Excel (.xlsx)** | Tables, data analysis | Microsoft Excel, Google Sheets |
| **Text (.txt)** | Simple text, maximum compatibility | Any text editor |
