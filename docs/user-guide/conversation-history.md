## What You See in History

Each conversation displays:
- **Title** - Auto-generated from first message
- **Date/Time** - When started
- **Preview** - First few words
- **Status** - Active or archived

---

## Step-by-Step: Reopening a Conversation

**1. Browse or Search**
- Scroll through history, OR
- Use search box to find specific conversations

**2. Click the Conversation**
- Full chat reopens
- All previous messages visible

**3. Continue Chatting**
- Type new messages
- NA remembers the context
- Add to the existing conversation
