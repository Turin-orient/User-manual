## File Size Limits

- **Maximum per file:** 10-25 MB (varies by organization)
- **Multiple files:** Can upload several at once
- **Total session limit:** Check with your admin

:::tip Large Files
For files over the limit, try:
- Compressing PDFs
- Splitting into smaller parts
- Uploading one section at a time
:::

## Step-by-Step: Uploading a Document

### Method 1: Click to Upload

**1. Locate Upload Button**
- Look for üìé (paperclip icon) in the chat input box
- OR look for "Upload" or "Attach" button

[Screenshot: Upload button highlighted]

**2. Click the Button**
- Click the paperclip/upload icon
- File browser opens

**3. Select Your File(s)**
- Navigate to the file location on your computer
- Click the file(s) you want to upload
- Hold Ctrl (Cmd on Mac) to select multiple files
- Click "Open"

[Screenshot: File browser window]

**4. Wait for Upload**
- Progress bar appears
- File name shows in chat when complete
- ‚úÖ Green checkmark indicates success

[Screenshot: Upload progress and completion]

---

### Method 2: Drag and Drop

**1. Open File Explorer/Finder**
- Locate your file(s)

**2. Drag to Chat Window**
- Click and hold the file
- Drag it over the Neurond Assistant window
- Drop zone highlights

**3. Release to Upload**
- Let go of the mouse button
- Upload begins automatically

[Screenshot: Drag and drop in action]

## After Upload: What to Do

### Give NA Instructions

Once uploaded, tell NA what to do with the file:

**For data extraction:**
```
"Extract the invoice number, date, vendor name, and total amount 
 from this invoice"
```

**For summarization:**
```
"Summarize the key points from this report"
```

**For specific questions:**
```
"According to this document, what is the project timeline?"
```

[Screenshot: Document uploaded with instructions being typed]

## Working with Multiple Documents

### Upload Multiple Files at Once

**To select multiple:**
1. Click upload button
2. Hold **Ctrl** (Windows) or **Cmd** (Mac)
3. Click each file you want
4. Click "Open"

**Good for:**
- Processing batch invoices
- Comparing documents
- Extracting data from multiple sources

**Example:**
```
[Upload 10 invoices]

"Create a table extracting invoice number, date, vendor,  
 and amount from all these invoices"
```

[Screenshot: Multiple files uploaded]

## Common Upload Tasks

### Task 1: Invoice Processing

**What to upload:** Invoice PDFs or photos

**What to ask:**
```
"Extract all invoice data into a table with columns:  
 Invoice Number, Date, Vendor, Amount, Due Date"
```

### Task 2: Resume Screening

**What to upload:** Candidate resumes (PDF/Word)

**What to ask:**
```
"Summarize each candidate's education, years of experience,  
 and key technical skills in a comparison table"
```

### Task 3: Report Summarization

**What to upload:** Long report (PDF/Word)

**What to ask:**
```
"Summarize this 40-page report focusing on:  
 1. Key findings  
 2. Recommendations  
 3. Budget implications"
```

### Task 4: Form Data Extraction

**What to upload:** Scanned forms (images or PDF)

**What to ask:**
```
"Extract the following fields from this application form:  
 Name, Address, Phone, Email, Date of Birth"
```

### File Won't Upload

- ‚ùå File too large
- ‚ùå Unsupported file type
- ‚ùå Network connection issue

**Solutions:**
1. ‚úì Check file size and type
2. ‚úì Try compressing the file
3. ‚úì Check internet connection

### Upload Freezes or Times Out

**Solutions:**
1. ‚úì Check internet speed
2. ‚úì Try smaller file first
3. ‚úì Close other browser tabs
4. ‚úì Try again in a few minutes

### NA Can't Read the Document

**Solutions:**
1. ‚úì Ensure text is not scanned image (use OCR-capable PDF)
2. ‚úì Check if file is password-protected (remove protection)
3. ‚úì Convert to standard format (PDF or Word)

## Security and Privacy

### Uploaded File Privacy

:::info Data Protection
- ‚úÖ Files are processed securely within company systems
- ‚úÖ Not shared externally
- ‚úÖ Subject to company data retention policies

### Sensitive Documents

**Before uploading, ask yourself:**
- Is this document work-related?
- Am I authorized to have this data processed?
- Does it contain customer PII? (check company policy)

**When in doubt, ask your supervisor or IT department.**
